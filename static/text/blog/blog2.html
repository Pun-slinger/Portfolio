An image is defined as a visual representation of something. An image is usually used to explain text content that should be used sparingly in order to enhance or expand ono the content while adding visual interest to the web page. By adding graphics like tables, pie charts, infographics, or simply a photo, you can help users to understand your content better when alone is unable to convey a clear idea.

For example, you're having a presentation about photography and at some point, you went into ways of improving your photos like Rule of Thirds and explains to others all about it. However, if no images was used, the audience would have absolutely no idea how an image with Rule of Thirds would look like nor be convinced that Rule of Thirds would improve a photo. Now you know how you should use images for creating multimedia content, you should also know that there are 2 types of images and knowing how these types works will allow you to use images more effectively in the process of creating.

  • Bitmap:
    • A simple matrix or grid of dots and color information.
    • Produce good quality images where shading and detail are needed, meaning they're best suitable for realistic photos or images that required high level of details
    • Higher detail also means bitmap required more memory space and bigger file size.
  • Vector:
    • Made up of objects such as straight lines, curves or shapes, and each portion of the image is represented mathematically.
    • Because vector images is made of lines and shapes, they're more suitable for multimedia content that involved 3d animations, graphic shapes, etc... like cartoons and symbols.
    • Portion of image is represented mathematically means vector images are able to be zoomed in, scaled, and resized, something bitmap is unable to do and results in decrease in quality.
    • Smaller file size, perfect for web content that require less downloading time and faster render.
